# Assets

Imported raw assets live here. Keep this folder focused on source art, audio, and import sidecars generated by Godot.

Project-authored gameplay resources belong in `res://game/resources/...`, not here. Preserve third-party pack layouts where practical so future asset updates are easier to apply.

Current asset areas:

- `art/`: characters, tilesets, UI art, and effects.
- `audio/`: reserved for music and sound effects.
