# Art Assets

Raw visual assets are grouped here by use.

- `characters/`: player, NPC, enemy, monster, and creature spritesheets from source packs.
- `tilesets/`: terrain, town, and world-building tiles.
- `ui/`: UI textures, fonts, and the Godot theme resource.
- `effects/`: visual effects and particles source art.

Do not add gameplay scripts or authored actor/item data here. Create game-facing resources under `res://game/resources/...` that reference these files.
